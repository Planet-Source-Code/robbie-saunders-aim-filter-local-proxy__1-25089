How to Configure AIM 4.7 Beta:
 - make sure you have at least aim version 4.7 (http://www.aol.com/aim/winbeta.html)
 - open your preferences
 - click on Sign On/Off on the menu
 - then click Connection (towards bottom right)
 - change the server (`login.oscar.aol.com`) to `localhost` or `127.0.0.1`
 - change the port if you want [optional, but required to send files]
 - click ok, make sure aim filter is running, and relogin
_____________________________________________________________________

How to Configure any other AIM:
 - open your preferences
 - click the Connection Tab
 - change the server (`login.oscar.aol.com`) to `localhost` or `127.0.0.1`
 - change the port if you want [optional, but required to send files]
 - click ok, make sure aim filter is running, and relogin
